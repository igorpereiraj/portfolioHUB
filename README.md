# portfolioHUB - Portfólio Profissional Digital 🌐

Este repositório foi desenvolvido como parte do **Desafio de Entrega Intermediária**, com o objetivo de centralizar e organizar projetos acadêmicos e pessoais, demonstrando competências em versionamento, documentação e boas práticas de desenvolvimento.

## 📂 Estrutura do Repositório

Seguindo as diretrizes do planejamento, o repositório está organizado da seguinte forma:

| Diretório | Descrição |
| :--- | :--- |
| `academicos/` | Projetos e exercícios desenvolvidos durante a trajetória estudantil. |
| `pessoais/` | Projetos de iniciativa própria, experimentos e estudos independentes. |
| `documentacao/` | Arquivos auxiliares, roteiros de apresentação e materiais do desafio. |
| `client/` | Código fonte da interface do usuário (Frontend). |
| `server/` | Lógica de servidor e integrações (Backend). |
